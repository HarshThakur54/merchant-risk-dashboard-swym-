# Merchant Risk Dashboard

A simple, modern, and responsive dashboard built to identify at-risk merchants and recommend actionable next steps based on churn signals. Created for the CX AI-Proficiency Build Round.

## Project Details
- **No Build Step:** This project is built using 100% Vanilla HTML, CSS, and JavaScript. 
- **Dynamic In-Memory Data:** All merchant data is randomly generated dynamically on the client-side (via `app.js`) to simulate an API payload with robust edge case testing (like enterprise downgrades and renewal urgencies).
- **V1 Specification:** Please see `v1_spec.md` in this repository for the full details on the data schema and the logic behind the churn signals and recommendations.
